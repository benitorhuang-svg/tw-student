# 產品規格書 (Product Specification) - 2026-03-10 更新

## 1. 產品定位與精神
本專案堅持 **「規格驅動開發 (SDD)」** 與 **「資料治理透明 (Data Governance)」**。
作為一個以台灣教育數據為核心的分析平台，我們不僅提供地理視覺化，更強調透過「比較」與「趨勢」產生決策洞察。

---

## 2. 五大主題分頁：圖表設計規格

### 2.1 概況總覽 (Overview) - 【全台大局觀】
*   **[Core] 全台學制發展面積圖 (Stacked Area Chart)**
    *   **資料維度**：X 軸為學年度 (107-114)，Y 軸為總學生數，系列為教育階段 (幼、國小、國中、高、大)。
    *   **視覺表現**：使用柔和漸層堆疊，強調各階段受少子化波擊的時間偏移性。
    *   **分析目標**：觀察教育資源需求如何從基礎教育往高等教育傳遞。
*   **[Core] 地區量體矩形圖 (Treemap)**
    *   **資料維度**：一級為區域 (北中南東)，二級為縣市，面積為目前學生數。
    *   **視覺表現**：依學生總量深度著色。

### 2.2 區域分析 (Regional) - 【跨域結構對照】
*   **[Refine] 區域內縣市消長排行 (Bar Chart with Delta)**
    *   **分析目標**：比較區域內縣市的「年變動率」，找出人口流動的吸引極 (Attraction Pole)。
*   **[New] 公私立平衡蝴蝶圖 (Butterfly Chart)**
    *   **分析目標**：左右對稱顯示公、私立佔比。

### 2.3 縣市分析 (County) - 【鄉鎮落差下鑽】
*   **[Highlight] 鄉鎮發展四象限矩陣 (Quadrant Scatter Plot)**
    *   **X 軸**：鄉鎮總學生數 (規模)
    *   **Y 軸**：年變動率 (增長趨勢)
    *   **輔助線**：縣市平均線 (十字交叉)。
    *   **象限定義**：
        1.  **右上**：核心成長區 (核心地位且持續擴張)。
        2.  **左上**：發展新星區 (規模尚小但快速成長)。
        3.  **左下**：偏鄉萎縮區 (低總量且持續失血)。
        4.  **右下**：穩定飽和區 (大體量但成長趨緩)。

### 2.4 各校分析 (Schools) - 【清單與分佈體檢】
*   **[Refine] 學校規模動態直方圖 (Histogram)**
    *   **分析目標**：分析該地區學校是否過度偏向「巨型校」或「極小校」，判斷資源分佈均勻度。
*   **[Iterative] 清單內嵌火花線 (Sparklines)**
    *   **視覺表現**：在學校資料表格內，每列顯示近五年趨勢微縮圖。

### 2.5 單校分析 (School Focus) - 【精準指標對比】
*   **[Core] 基準線趨勢圖 (Benchmark Trend)**
    *   **分析目標**：在學校趨勢圖中，加入「所在鄉鎮平均」與「所在縣市平均」作為背景對照線。
*   **[New] 學制作業與定位圖 (PR Indicator)**
    *   **分析目標**：呈現該校在全縣同學制中的百分等級 (PR)。

## 3. 視覺優化與互動規格 (P0)

### 3.1 分頁列互動 (AtlasTabs)
*   **規格**：導入液態背景滑動指示器 (Sliding Indicator)。
*   **精神**：提供流暢的操作回饋，增加介面精密感。

### 3.2 趨勢圖進場動畫
*   **規格**：SVG Stroke-dashoffset 動畫，實現「繪製感」進場。

### 3.3 地圖與分頁聯動
*   **規格**：點選地圖縣市/鄉鎮時，系統需自動判斷並切換至對應分頁。
*   **精神**：隨者點選深度自動切換視角，減少手動切換分頁的次數。
