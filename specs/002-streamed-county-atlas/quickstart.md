# 快速驗證：縣市細節串流與版型深化

## 本機執行

1. 在 `frontend/` 執行 `npm run data:refresh`。
2. 在 `frontend/` 執行 `npm run dev`。
3. 另開終端執行 `npx playwright test`。

## 驗證情境

### 情境一：首頁只載入摘要

1. 開啟首頁。
2. 檢查 Network。
3. 確認首頁沒有同步請求所有 `counties/*.json` 細節檔。

### 情境二：深連結還原

1. 以帶有 `year`、`county`、`township`、`level` 的網址直接進站。
2. 確認頁面最終回到指定縣市與鄉鎮。
3. 確認鄉鎮切片與縣市細節檔在進站後被補載。

### 情境三：縣市切換載入細節與界線

1. 首頁先維持全台模式。
2. 點擊任一縣市。
3. 確認畫面顯示縣市細節載入狀態。
4. 確認該縣市鄉鎮地圖、排行與學校清單更新。

### 情境四：異常註記顯示

1. 切換到含異常註記的縣市或學校。
2. 確認 `待確認` 狀態、缺年度註記或其他 `dataNotes` 有顯示。

### 情境五：桌機與行動版版型

1. 桌機寬度下確認控制區、地圖、摘要、排行與學校清單版位清楚。
2. 行動版寬度下確認模組可順序重排且保持可操作。
