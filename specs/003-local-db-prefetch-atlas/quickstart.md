# 快速驗證：本地資料庫預抓與可觀測縣市 Atlas

## 本機執行

1. 在專案根目錄執行 `npm run data:refresh`。
2. 在專案根目錄執行 `npm run dev`。
3. 另開終端執行 `npm run test:e2e`。

## 驗證情境

### 情境一：首頁不載全量細節

1. 開啟首頁。
2. 檢查 Network。
3. 確認沒有一次下載全部 `counties/*.json`。

### 情境二：hover 預抓

1. 將滑鼠移到某縣市上。
2. 檢查 Network 與來源觀測區。
3. 確認只有該縣市相關切片被補載。

### 情境三：本地快取命中

1. 點進某縣市後再回到全台。
2. 再次進入同一縣市。
3. 確認來源觀測區中的快取命中增加。

### 情境四：學校表格排序與匯出

1. 進入任一縣市。
2. 在學校表格點擊 `學生數`、`年增減`、`狀態` 欄位排序。
3. 點擊 `匯出 CSV` 並確認檔案可下載。
